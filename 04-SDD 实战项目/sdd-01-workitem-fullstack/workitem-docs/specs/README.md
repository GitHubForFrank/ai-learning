## 四、阅读路径建议

**新人入门**：按 01 → 02 → 03 → 04 顺序阅读，了解完整 SDD 流程。

**编码时查阅**：需要 DB 命名规则 → `91-编码规范.md Part A`；需要错误码分段 → `92-接口治理.md Part B`。

**评审 / 上线前检查**：对照 `90-工程约束.md` 各 Part 的禁用清单逐项检查。
