## 使用建议

- **入门阶段**：先刷模块 1、2、3，建立基础认知
- **进阶阶段**：模块 4、5、6、7、8，覆盖架构设计核心
- **冲刺阶段**：模块 9、10、11，应对深度追问和设计题
- **模拟面试**：选 3 题基础 + 2 题架构 + 1 题开放题，限时 45 分钟回答
