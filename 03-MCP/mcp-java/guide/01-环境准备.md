## 1. 环境准备

### 1.1 前置：安装 JDK

要求 **JDK 17+**（Spring Boot 3 最低要求）。推荐 21 LTS。

| 平台 | 推荐方式 |
|------|---------|
| 全平台 | [Eclipse Temurin](https://adoptium.net/)（Adoptium 基金会免费分发）/ [Azul Zulu](https://www.azul.com/downloads/) |
| Windows | `winget install EclipseAdoptium.Temurin.21.JDK` |
| macOS | `brew install temurin` |
| Linux | `apt install temurin-21-jdk` 或 SDKMAN |
| 多版本管理 | [SDKMAN](https://sdkman.io/)（`sdk install java 21-tem`） |

**验证**：

```bash
java -version          # 应 ≥ 17
javac -version
```

### 1.2 前置：安装 Maven

Spring AI 官方样板用 Maven（也可改 Gradle，但本文以 Maven 为例）：

| 平台 | 推荐方式 |
|------|---------|
| Windows | `winget install Apache.Maven` |
| macOS | `brew install maven` |
| Linux | `apt install maven` 或 SDKMAN |

**验证**：

```bash
mvn -v
```

### 1.3 从零创建项目

最快路径是用 Spring Initializr：

```bash
# 方式 A：浏览器
# 打开 https://start.spring.io/
# 选 Project = Maven、Language = Java、Spring Boot = 3.x
# Dependencies 搜索并添加 "MCP Server"
# 点 Generate 下载 zip，解压后 cd 进去

# 方式 B：命令行
curl https://start.spring.io/starter.zip \
  -d dependencies=spring-ai-starter-mcp-server-webmvc \
  -d type=maven-project \
  -d language=java \
  -d bootVersion=3.3.5 \
  -d javaVersion=17 \
  -d groupId=com.example \
  -d artifactId=my-mcp-server \
  -d name=my-mcp-server \
  -o my-mcp-server.zip

unzip my-mcp-server.zip -d my-mcp-server
cd my-mcp-server      # 后续命令都在这里跑
```

### 1.4 Spring AI MCP Server Starter

Spring AI 提供多个 Starter，按传输/部署形态选（二选一，互斥）：

| Starter | 适用 | 典型场景 |
|---------|------|---------|
| `spring-ai-starter-mcp-server` | 纯 Stdio | 本地工具、桌面集成 |
| `spring-ai-starter-mcp-server-webmvc` | Streamable HTTP（Servlet） | 传统 Spring Boot 服务 |
| `spring-ai-starter-mcp-server-webflux` | Streamable HTTP（响应式） | 高并发、流式长任务 |

**包来源**：[Maven Central](https://search.maven.org/artifact/org.springframework.ai/spring-ai-starter-mcp-server-webmvc)

**手动加依赖**（若未用 Initializr）：

```xml
<!-- pom.xml 中 <dependencies> 内 -->
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-starter-mcp-server-webmvc</artifactId>
</dependency>
```

Spring AI 目前还没进 Spring Boot BOM，**必须**在 `<dependencyManagement>` 里引入 Spring AI BOM 统一版本：

```xml
<dependencyManagement>
  <dependencies>
    <dependency>
      <groupId>org.springframework.ai</groupId>
      <artifactId>spring-ai-bom</artifactId>
      <version>1.0.0</version>     <!-- 以 https://spring.io/projects/spring-ai 最新稳定版为准 -->
      <type>pom</type>
      <scope>import</scope>
    </dependency>
  </dependencies>
</dependencyManagement>
```

### 1.5 拉取依赖

```bash
# 当前目录是项目根（含 pom.xml）
mvn dependency:resolve
```

首次会从 Maven Central 下载所有 jar 到 `~/.m2/repository/`。**中国大陆**建议先配置阿里云镜像（`~/.m2/settings.xml`）：

```xml
<settings>
  <mirrors>
    <mirror>
      <id>aliyun</id>
      <mirrorOf>central</mirrorOf>
      <url>https://maven.aliyun.com/repository/public</url>
    </mirror>
  </mirrors>
</settings>
```

### 1.6 验证安装

```bash
mvn clean compile
```

看到 `BUILD SUCCESS` 即依赖与编译环境都 OK。

### 1.7 目标项目结构

```
my-mcp-server/
├── pom.xml
└── src/main/
    ├── java/com/example/mcp/
    │   ├── McpServerApplication.java      # Spring Boot 入口
    │   └── tools/
    │       ├── WeatherTools.java          # @McpTool 方法集合
    │       └── FileTools.java
    └── resources/
        └── application.yml                # MCP 协议/端口/元数据配置
```

### 1.8 application.yml

```yaml
spring:
  ai:
    mcp:
      server:
        name: my-mcp-server
        version: 0.1.0
        protocol: STREAMABLE    # 或 STDIO
server:
  port: 8080
```
