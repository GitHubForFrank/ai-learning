# MCP（Model Context Protocol）Agent 面试题库

本面试题库涵盖 MCP 协议的各个层次，从基础概念到多 Agent 编排实战，共 **9 个专题、180+ 道题目**。

题目来源：
- 笔记提炼：03-MCP 学习笔记（协议规范、开发指南、设计规范、安全实践、生态集成）
- 实战总结：03-MCP 实战项目（6 语言 × 12 个 MCP Server 项目）
- 网络整理：知乎、CSDN、Docsity 等平台高频面试题

## 使用建议

- **快速突击**：先看每章的"高频题"（标注 ★★★），掌握核心概念
- **系统准备**：按目录顺序通读，每章末尾有"自检清单"
- **模拟面试**：使用第 09 章的场景题，找同伴进行模拟问答
- **实战验证**：结合 `03-MCP 实战项目/` 中的代码，理解理论如何落地
